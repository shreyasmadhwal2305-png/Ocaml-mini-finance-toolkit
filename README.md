# OCaml Mini Finance Toolkit

A beginner-friendly, console-based finance toolkit written in **OCaml**.  
This project is designed to help retain OCaml fundamentals

The code is intentionally kept:
- simple
- clean
- single-file
- beginner-readable

---

## Features

- Simple Interest calculator
- Compound Interest calculator
- SIP (Systematic Investment Plan) calculator
- Net Worth calculator

---

## Concepts Used

- Pure functions
- Float and integer arithmetic
- Lists
- `List.fold_left`
- Function composition
- Console-based execution using `utop`

No advanced modules, libraries, or functors are used.

---
